I wrote two Python scripts to test latency and consistency for Cloud Datastore. I used to requests library to send all of my http requests. For latency, I sent 10 requests per server per endpoint and calculated per endpoint and server averages. For consistency, I ran 100 iterations of a loop where I would generate a unique username using uuid, then register the user at one server and call list to the other server immediately. If the username wasn't on the list, I noted it, and then waited some time to request again to see if eventual consistency was met. I also switched up the servers I was registering and listing to and did the same exact experiment with them.
